# LMS Backend - CodeIgniter 4

Αυτό είναι το backend του συστήματος διαχείρισης βιβλιοθήκης (LMS), υλοποιημένο σε CodeIgniter 4.

Το backend παρέχει ένα REST API που επιτρέπει τη διαχείριση χρηστών, βιβλίων και συγγραφέων, με authentication μέσω CI Shield.

## Λειτουργίες που έχουν υλοποιηθεί

**Χρήστες (Users)**

- [✅] Εγγραφή χρήστη (Register)
- [✅] Σύνδεση χρήστη (Login) με δημιουργία Token
- [✅] Διαγραφή χρήστη (Delete) με Αuthentication token
- [✅] Αναζήτηση χρηστών (Get) με Αuthentication token
- [✅] Ενημέρωση χρήστη (Update) με Αuthentication token

**Συγγραφείς (Authors)**

- [✅] Προσθήκη νέου συγγραφέα (Create)
- [✅] Εμφάνιση όλων των συγγραφέων (Get)
- [✅] Ενημέρωση συγγραφέα (Update)
- [✅] Διαγραφή συγγραφέα (Delete) (Δεν μπορεί να διαγραφεί αν έχει βιβλία)

**Βιβλία (Books)**

- [✅] Προσθήκη νέου βιβλίου (Create)
- [✅] Εμφάνιση όλων των βιβλίων ενός χρήστη (Get)
- [✅] Ενημέρωση βιβλίου (Update)
- [✅] Διαγραφή βιβλίου (Delete)
- [✅] Μη δυνατότητα εισαγωγής βιβλίου με ίδιο τίτλο και ίδιο συγγραφέα

**Ασφάλεια & Authentication**

- [✅] Χρήση token-based authentication μέσω CodeIgniter Shield
- [✅] Μόνο authenticated χρήστες έχουν πρόσβαση στις λειτουργίες
- [✅] Οι χρήστες βλέπουν μόνο τα δικά τους δεδομένα (βιβλία, συγγραφείς)

## Λειτουργίες που δεν έχουν υλοποιηθεί

- [x] Δεν έχει υλοποιηθεί σύστημα αλλαγής κωδικού χρήστη
- [x] error handling για καλύτερη ανατροφοδότηση
- [x] Ενω φαίνεται να διαγράφεται ο χρήστης μέσω postman, στη βάση δεδομένων φαίνεται να υπάρχει.

## Βάση Δεδομένων

Η βάση δεδομένων είναι σε MySQL. Περιλαμβάνει τους εξής πίνακες:

- `users`
- `authors`
- `books`
- `settings`
- `migrations`
